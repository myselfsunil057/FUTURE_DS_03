## 🎥 Demo Summary

This project showcases an interactive **4-page E-Commerce Intelligence Dashboard** built using Power BI, designed to analyze over **100M+ customer interaction events** across a multi-category online retail platform (Oct–Nov 2019).

The dashboard simulates a real-world analytics system used to monitor **customer behavior, funnel efficiency, revenue performance, and category-level profitability**.

---

## 🧠 What This Dashboard Demonstrates

This is not just a reporting tool — it is a **full-funnel decision intelligence system** that helps answer:

- How users move from **product discovery → cart → purchase**
- Where revenue is being lost in the funnel
- Which categories drive the most value vs engagement
- How campaigns impact revenue spikes and user behavior
- Where operational improvements can unlock growth

---

## 📊 Dashboard Structure

### 🏠 Landing Page (Executive Overview)
- High-level KPIs (Revenue, Orders, Conversion Rate)
- Quick navigation across all analytical pages
- Business snapshot for leadership decision-making

---

### 📈 E-Commerce Insights Hub
- Revenue and order performance tracking
- Month-over-month growth analysis
- Conversion rate and revenue efficiency metrics

---

### 👁️ View Insights (Attention Analysis)
- User browsing behavior patterns
- Product view distribution across categories

---

### 🛒 Cart Insights (Conversion Behavior)
- Cart addition trends
- Cart abandonment analysis
- Cart → Order conversion efficiency breakdown

---

## 🔍 Key Analytical Outcome

The dashboard reveals a **critical funnel imbalance**:

- 🔴 96.2% users drop before adding to cart  
- 🟠 58% cart abandonment rate  
- 🟢 Only ~1.9% overall conversion rate  

👉 Insight: The biggest business issue is **attention-to-intent failure**, not post-intent checkout.
